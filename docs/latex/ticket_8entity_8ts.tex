\doxysubsection{Подробное описание}
Entity тикета. 

Представляет тикет в системе техподдержки. Содержит информацию об авторе, назначенном исполнителе, заголовок, сообщения и статус тикета.

\begin{DoxyRemark}{Прим.}
Это сущность доменного слоя. Может содержать несколько сообщений и использоваться как агрегат для управления сообщениями и статусом. 
\end{DoxyRemark}
